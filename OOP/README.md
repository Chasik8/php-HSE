# PHP OOP E-Commerce Project

Учебный проект, демонстрирующий принципы ООП на примере интернет-магазина.

## Требования
Выполнены все пункты задания:
- [x] Минимум 5 классов (AbstractProduct, PhysicalProduct, DigitalProduct, Cart, Anonymous)
- [x] Интерфейсы (ProductInterface) и Трейты (LoggerTrait)
- [x] Магические методы (__construct, __destruct, __toString, __debugInfo, __clone, __sleep, __wakeup)
- [x] Наследование и Полиморфизм
- [x] Анонимные классы и Final
- [x] Сериализация и Клонирование

## Запуск
1. Склонируйте репозиторий.
2. Запустите `php index.php` в консоли.
